## Quick Start

The curriculum is available in three ways:

1. **SPARQL Endpoint**: Query live data at `https://national-curriculum-for-england-sparql-*.run.app/national-curriculum-for-england/sparql`
2. **JSON Downloads**: Pre-generated files from GitHub Releases
3. **TTL Files**: Raw RDF data from this repository

See [API Examples](api-examples.md) for code samples in Python, JavaScript, and curl.
